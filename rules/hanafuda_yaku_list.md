# 花合わせ 役一覧

---

## 光役

| 役名 | 条件 | 点数 |
| :---- | :---- | :---- |
| 五光 | 光札5枚すべて（松に鶴・桜に幕・芒に月・雨に人・桐に鳳凰） | 50点 |
| 四光 | 雨に人を含まない光4枚 | 40点 |
| 雨四光 | 松に鶴・芒に月・桐に鳳凰・雨に人の4枚 | 40点 |
| 三光 | 松に鶴・芒に月・桐に鳳凰の3枚 | 30点 |

---

## 短冊役

| 役名 | 条件 | 点数 |
| :---- | :---- | :---- |
| 赤短 | 松・梅・桜の短冊3枚（松に赤短・梅に赤短・桜に赤短） | 30点〜 |
| 青短 | 牡丹・菊・紅葉の短冊3枚（牡丹に青短・菊に青短・紅葉に青短） | 30点〜 |

**加点ルール（赤短・青短共通）：** 規定の3枚に加え、短冊札を追加で取るたびに **\+10点**。

---

## その他の役

| 役名 | 条件 | 点数 |
| :---- | :---- | :---- |
| 猪鹿蝶 | 萩に猪・紅葉に鹿・牡丹に蝶の3枚 | 15点 |
| 月見酒 | 芒に月＋菊に盃 | 10点 |
| 花見酒 | 桜に幕＋菊に盃 | 10点 |
| カス | カス13枚以上（菊に盃はカス1枚として加算） | 10点〜 |
| フケ | 取り点が手番の基準点以下 | 80点 |

**カス加点ルール：** 13枚を超えるごとに **\+10点**。菊に盃はカス1枚として計算に含む。

**フケについて：** 他の役と同時に成立した場合、その役点にフケの80点が加算されます。

---

## 特殊ルール

| 名称 | 条件 | 効果 |
| :---- | :---- | :---- |
| 雨流れ | 1人が11月の柳札4枚すべてを取得 | 全員のでき役点が無効（フケを除く） |
| 三役流れ | 3人全員にフケ以外のでき役が成立 | 全員のでき役点が無効（フケを除く） |
