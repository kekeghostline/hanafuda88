# 花合わせ ルール説明

## 概要

3人で行うカードゲームです。**12回戦**の合計点が最も高い人が勝利します。

---

## 準備・配札

- 花札48枚を使用します  
- 各プレイヤーに手札を配り、残りは山札として中央に置きます

---

## 進行順序

各回、手番は **親 → 胴二 → ビキ** の順に行います。 親は毎回交代し、12回戦で全員が均等に親を担当します。

### 手番の流れ

1. 手札から1枚出し、場の札と合わせて取る（合わなければ場に置く）  
2. 山札から1枚めくり、場の札と合わせて取る（合わなければ場に置く）  
3. 手札と山札がなくなるまで続けます

---

## 得点

各回の得点は **札点 ＋ でき役点** で計算します。

### 札点

取った札それぞれに定められた点数の合計です。

### でき役

取り終えた時点で成立している役の点数が加算されます。 途中で役ができても**そのままゲームを続行します**（こいこいはありません）。

---

## 特殊ルール

### 三役流れ

3人全員にでき役が成立した場合、その回のでき役点は**全員無効**になります。 （札点はそのまま有効です）

### 雨流れ

11月の柳札（雨札）4枚すべてを1人が取った場合、その回のでき役点は**全員無効**になります。 （札点はそのまま有効です）

### フケ

あえて点を取らない特殊な役です。

- 手番ごとに**基準点**が設定されており、その点以下であれば成立します  
- 三役流れ・雨流れが発生しても**フケは無効になりません**

---

## 勝利条件

12回戦終了後、合計点が最も高いプレイヤーの勝利です。

---

## ゲーム基本情報

| 項目 | 内容 |
| :---- | :---- |
| 使用枚数 | 48枚 |
| 手札（各自） | 7枚 |
| 場札 | 6枚 |
| 山札 | 21枚 |
| 回戦数 | 12回戦 または 6回戦 |
| 基礎点 | 80点 |

**基礎点について：** 各回の得点から80点を差し引いた差分が実際の得点になります。80点ちょうどの場合は ±0点 です。

---

## 手番・フケ基準点

| 手番 | 名称 | フケ基準点 |
| :---- | :---- | :---- |
| 1番手 | 親 | 15点以下 |
| 2番手 | 胴二 | 20点以下 |
| 3番手 | ビキ | 25点以下 |
