# 花合わせ ルール説明

## ゲームの流れ

### 回戦の進行

3人で行い、**12回戦（または6回戦）** の合計点が最も高いプレイヤーの勝利です。

毎回、手番の順番は **親 → 胴二 → ビキ** で進めます。親は毎回交代します。

### 手番の手順

1. 手札から1枚出し、場の札と合わせて取る（合わなければ場に置く）  
2. 山札から1枚めくり、場の札と合わせて取る（合わなければ場に置く）

手札と山札がなくなるまで続け、その回を終了します。

---

## 得点計算

各回の得点は **「札点 ＋ でき役点 － 基礎点（80点）」** で計算します。

- **札点：** 取った札それぞれに定められた点数の合計  
- **でき役点：** 成立した役の点数の合計

途中で役ができても**そのままゲームを続行します**（こいこい・終了宣言はありません）。

---

## 特殊ルール

### 三役流れ

3人全員にでき役が成立した場合、その回の**でき役点は全員無効**になります。 （札点はそのまま有効です）

### 雨流れ

11月の柳札（雨札）4枚すべてを1人が取った場合、その回の**でき役点は全員無効**になります。 （札点はそのまま有効です）

### フケ

あえて点を取らないことで成立する特殊な役です。

- 手番ごとに設定された**基準点以下**の得点であれば成立します  
- 三役流れ・雨流れが発生しても**フケは無効になりません**

| 手番 | 名称 | フケ成立条件 |
| :---- | :---- | :---- |
| 1番手 | 親 | 15点以下 |
| 2番手 | 胴二 | 20点以下 |
| 3番手 | ビキ | 25点以下 |

---

## 勝利条件

12回戦（または6回戦）終了後、**合計点が最も高いプレイヤーの勝利**です。

---

## ゲーム基本情報

| 項目 | 内容 |
| :---- | :---- |
| 使用枚数 | 48枚 |
| 手札（各自） | 7枚 |
| 場札 | 6枚 |
| 山札 | 21枚 |
| 回戦数 | 12回戦 または 6回戦 |
| 基礎点 | 80点 |

**基礎点について：** 各回の得点から80点を差し引いた差分が実際の得点になります。80点ちょうどの場合は ±0点 です。
